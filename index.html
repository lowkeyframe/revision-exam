1. Histoire de l’informatique

  • Informatique : Science du traitement numérique de l’information.
  • Grandes périodes :
    • Antiquité : Premiers outils de calculs (Exemple : boulier)
    • Renaissance : Machine analytique de Babbage, Métier à Tisser de Jacquard
    • Années 1930-40 : Alan Turing formalise la base théorique de l’ordinateur
    • Années 1940-50 : Premiers ordinateurs électroniques (ENIAC, UNIVAC)
    • Années 1970-80 : Micro-ordinateurs, premiers systèmes personnels
    • Années 1990 : Internet commercialisé
    • Années 2000-2010 : Web 2.0, réseaux sociaux, smartphones
    • Aujourd’hui : IA, cloud, big data

2. Composantes d’un ordinateur

  • CPU : Cerveau de l’ordinateur (cœurs, threads, cadence en GHz, cache)
  • RAM : Mémoire temporaire à court terme (8 Go minimum, 16 Go recommandé)
  • Disques :
    • HDD : mécanique, grande capacité, lent (jusqu’à 16 To)
    • SSD : électronique, rapide, capacité moyenne (jusqu’à 4 To)
  • Carte mère : Relie toutes les composantes
  • Carte graphique (GPU) : Affichage et traitement parallèle
  • PSU : Fournit l’énergie nécessaire
  • Boîtier : Protège et organise les composants

3. Logiciels et Systèmes d’exploitation

  • Systèmes d’exploitation : Windows (Microsoft), MacOS (Apple), Linux (open-source), Android (Google), iOS (Apple)
  • Pilotes (drivers) : Permettent la communication matériel/OS
  • Applications : Logiciels pour des tâches spécifiques

4. Formatage et Partitions

  • Formater = préparer un disque avec un système de fichiers
  • Formats :
    • exFAT : Compatible Mac/Windows (> 4 Go)
    • FAT32 : Pour < 4 Go
    • NTFS : Pour Windows
    • Mac OS étendu journalisé : Pour Mac
  • Partition = division logique d’un disque

5. Données et Préfixes

  • Bit = 0 ou 1
  • 1 octet = 8 bits
  • Préfixes :
    • Kilo (KB) = 1 000 octets
    • Méga (MB) = 1 000 000 octets
    • Giga (GB) = 1 000 000 000 octets
    • Tera (TB) = 1 000 000 000 000 octets

6. Programmation

  • Concept : Parler le langage de la machine pour exécuter des instructions
  • Langages :
    • Web : HTML, CSS, JavaScript, PHP
    • Généraliste : Python
    • Bas niveau : C
    • Applications exigeantes : C++, C#
    • Base de données : SQL
  • Concepts de base :
    • Variables : Conteneurs de données
    • Conditions : Si… alors (if… then)
    • Boucles : Répétitions (while, for each)
    • Fonctions : Blocs réutilisables
    • Événements : Déclenchement par l’utilisateur (clic, mouvement…)

7. Web et Réseaux

  • LAN : Réseau local
  • WAN : Réseau étendu
  • Internet : Réseau mondial
  • Web : Service pour afficher les pages via un navigateur
  • Domaine : Adresse lisible (ex. www.site.com)
  • Hébergement : Serveur où sont stockés les fichiers du site
  • Front-end : Interface visible (design, UX)
  • Back-end : Logique serveur, bases de données, requêtes
  • Full-stack : Fait les deux (front-end + back-end)

8. Git et GitHub

  • Git : Logiciel local de gestion de versions (pas besoin d’Internet), développé par Linus Torvalds
  • GitHub : Plateforme d’hébergement pour projets Git (collaboration en ligne)

9. Procédures Teams

  • Classer des fichiers par format → archiver en .zip → déposer sur Teams
  • Identifier les composantes d’un poste → via Propriétés de « Ce PC » ou Gestionnaire de périphériques
  • Réserver un local → via Teamup
  • Réserver du matériel → via formulaire en ligne

10. Formats de fichiers importants

  • Texte : .txt, .doc/.docx, .pdf, .csv
  • Image : .jpg, .png, .gif, .tif, .webp
  • Audio/Vidéo : .mp3, .wav, .flac, .mp4
  • Web : .html, .css, .js, .json, .md
  • Archives : .zip, .rar, .7z, .iso
