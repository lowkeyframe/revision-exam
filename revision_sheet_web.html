Feuille de Révision – Examen d’Introduction à l’Informatique

1. Histoire de l’informatique

  • Informatique : Science du traitement numérique de l’information.
  • Grandes périodes :
    • Antiquité : Premiers outils de calculs (Exemple : boulier)
    • Renaissance : « programme mécanique » avec la Machine analytique de Babbage et le Métier à Tisser de Jacquard
    • Années 1930-40 : Alan Turing formalise la base théorique de l’ordinateur.
    • Années 1940-50 : Premiers ordinateurs électroniques (ENIAC, UNIVAC).
    • Années 1970-80 : Micro-ordinateurs, premiers systèmes personnels.
    • Années 1990 : Internet commercialisé.
    • Années 2000-2010 : Web 2.0, réseaux sociaux, smartphones.
    • Aujourd’hui : IA, infonuagique, big data.

2. Composantes d’un ordinateur

  • CPU : Cerveau de l’ordinateur (cœurs, threads, GHz, cache).
  • RAM : Mémoire temporaire (8–16 Go recommandé).
  • Disques :
    • HDD : Grande capacité, lent (jusqu’à 16 To).
    • SSD : Rapide, capacité moyenne (jusqu’à 4 To).
  • Carte mère : Relie toutes les composantes.
  • Carte graphique (GPU) : Affichage, traitement parallèle.
  • PSU : Alimentation adaptée à la puissance requise.
  • Boîtier : Protège et ventile.

3. Logiciels et Systèmes d’exploitation

  • Systèmes d’exploitation : Windows, MacOS, Linux, Android, iOS
  • Pilotes (drivers) : Communication matériel/OS.
  • Applications : Logiciels pour des tâches spécifiques.

4. Formatage et Partitions

  • Formater = préparer un disque avec un système de fichiers.
  • Formats : exFAT, FAT32, NTFS, Mac OS étendu journalisé.
  • Partition = division logique d’un disque.

5. Données et Préfixes

  • Bit = 0 ou 1
  • 1 octet = 8 bits
  • Préfixes : KB, MB, GB, TB

6. Programmation

  • Concept : Parler le langage de la machine.
  • Langages : HTML, CSS, JavaScript, PHP, Python, C, C++, C#, SQL
  • Concepts de base : Variables, Conditions, Boucles, Fonctions, Événements

7. Web et Réseaux

  • LAN = réseau local
  • WAN = réseau étendu
  • Internet = infrastructure
  • Web = service via navigateur
  • Domaine, Hébergement, Front-end, Back-end, Full-stack

8. Git et GitHub

  • Git : Gestion de versions locale (créé par Linus Torvalds).
  • GitHub : Plateforme collaborative en ligne.

9. Procédures Teams

  • Classer fichiers → .zip → déposer sur Teams.
  • Identifier composantes → « Ce PC » ou Gestionnaire.
  • Réserver un local → Teamup.
  • Réserver matériel → formulaire.

10. Formats de fichiers importants

  • Texte : .txt, .doc/.docx, .pdf, .csv
  • Image : .jpg, .png, .gif, .tif, .webp
  • Audio/Vidéo : .mp3, .wav, .flac, .mp4
  • Web : .html, .css, .js, .json, .md
  • Archives : .zip, .rar, .7z, .iso
